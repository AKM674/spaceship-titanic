# Spaceship Titanic 🚀

A Kaggle competition project predicting which passengers were 
transported to another dimension using Machine Learning.

## Result
- **Kaggle Score**: 80.07% accuracy
- **Rank**: 910th

## Approach
- Exploratory Data Analysis — found CryoSleep passengers 
  transported 82% of the time
- Feature Engineering — extracted Deck, Side from Cabin column, 
  created TotalSpend, GroupSize, AgeBin
- Compared Random Forest vs XGBoost
- Hyperparameter tuning using GridSearchCV

## Tools Used
- Python, Pandas, Scikit-learn, XGBoost
- Kaggle Notebooks
